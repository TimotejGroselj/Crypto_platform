# Crypto_platform
## Podatki
Podatke za cene kovancev pridobivava iz spleta, ki ima omejeno število requestov. Ko poženete katerega izmed vmesnikov se bo po potrebi podatkovna baza sama ustvarila ter nastavila začetne vrednosti. Polega tega se bodo posodobila tudi cene kovancev (za vsak dan je ena cena in če cene za današnji dan ni se posodobi), kar vzame okoli 2.5 minute (zaradi omejitve requestov)


## Tekstovni vmesnik
Za uporabo spletnega vmesnika poženite main.py.

## Spletni vmesnik
Za uporabo spletnega vmesnika poženite app.py.
